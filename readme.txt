DA NAPKIN GOD ULTIMATE GAME of the EON III
------------------------------------------

Welcome to DNGUGotE3, this is a turn based fighting game all in batch, VBScript, and AutoHotkey, it comes complete with a launcher,
its own file format for saves, a card visualizer for the save files (to flex them to others!), and a registry file for registering the file format.

How to Start the game:
----------------------

If this IS your first time launching the game:
vvvvvvvvvvvvvvvvvvvvvvvvvvvvvvvvvvvvvvvvvvvvvv
Double click "setup.bat" in the root folder and wait for the installation
Then either double click the desktop shortcut if you made one, or go to /game-files/ and double click "open-l.vbs"

If this is NOT your first time (meaning you have set up the game):
vvvvvvvvvvvvvvvvvvvvvvvvvvvvvvvvvvvvvvvvvvvvvvvvvvvvvvvvvvvvvvvvvv
either double click /game-files/open-l.vbs or if you have a desktop shortcut, double click it
it will bring you to the launcher, when in the launcher choose what to open, or one of the other options.

If you just MOVED the game files:
vvvvvvvvvvvvvvvvvvvvvvvvvvvvvvvvv
re-run setup.bat to fix the paths

Paths & Directories:
--------------------
/game-files/: where all the game files are stored
/game-files/icons: where the icons for the files and the game are stored (holytext.ico and dngugote3.ico)
/game-files/saves: where all your save files are
/game-files/other: there is currently only the registry script in here
